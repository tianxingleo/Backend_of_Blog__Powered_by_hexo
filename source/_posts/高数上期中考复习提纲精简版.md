# 数学笔记整理

## 1. 基础三角恒等式与反三角函数关系

**(1)**

$$1 + \tan^2 x = \sec^2 x$$

$$1 + \cot^2 x = \csc^2 x$$

**(2)**

$$\arcsin x + \arccos x = \frac{\pi}{2}$$

$$\arctan x + \text{arccot } x = \frac{\pi}{2}$$

## 2. 重要极限

**(3)**

$$\lim_{x \to 0} (1 + x)^{\frac{1}{x}} = e$$

## 4. 高阶导数公式

**(5)**

$$(a^x)^{(n)} = a^x (\ln a)^n$$

$$(e^{ax})^{(n)} = a^n e^{ax}$$

$$\left(\frac{1}{ax+b}\right)^{(n)} = \frac{(-1)^n n! a^n}{(ax+b)^{n+1}}$$

$$[\sin(ax+b)]^{(n)} = a^n \sin\left(ax + b + \frac{n\pi}{2}\right)$$

$$(\sin x)^{(n)} = \sin\left(x + \frac{n\pi}{2}\right)$$

$$[\cos(ax+b)]^{(n)} = a^n \cos\left(ax + b + \frac{n\pi}{2}\right)$$

$$(\cos x)^{(n)} = \cos\left(x + \frac{n\pi}{2}\right)$$

$$[\ln(ax+b)]^{(n)} = \frac{(-1)^{n-1} a^n (n-1)!}{(ax+b)^n}$$

$$(x^\alpha)^{(n)} = \alpha(\alpha-1)\cdots(\alpha-n+1)x^{\alpha-n}$$

$$(x^n)^{(n)} = n!$$

## 5. 泰勒公式与展开

**(6) 泰勒公式 (带拉格朗日余项)**

$$f(x) = f(x_0) + \frac{f'(x_0)}{1!}(x-x_0) + \frac{f''(x_0)}{2!}(x-x_0)^2 + \cdots + \frac{f^{(n)}(x_0)}{n!}(x-x_0)^n + R_n$$

$$R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-x_0)^{n+1}$$

**(7) 常用麦克劳林展开式**

$$e^x = 1 + x + \frac{x^2}{2!} + \cdots$$

$$\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots$$

$$\cos x = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots$$

$$\ln(1+x) = x - \frac{x^2}{2} + \frac{x^3}{3} - \cdots$$

$$(1+x)^\alpha = 1 + \alpha x + \frac{\alpha(\alpha-1)}{2!}x^2 + \cdots$$

$$\sqrt{1+x} = 1 + \frac{1}{2}x - \frac{1}{8}x^2 + \cdots$$

$$\frac{1}{1+x} = 1 - x + x^2 - \cdots$$

$$\frac{1}{1-x} = 1 + x + x^2 + \cdots$$

**(8) 概念辨析**

- **驻点、极值点、间断点、不可导点**：指 **数** ($x$ 值)。
- **拐点**：指 **点** (坐标 $(x, y)$)。

## 6. 曲率与中值定理

**(9) 曲率与曲率圆**

- **曲率**: $k = \frac{|y''|}{(1+y'^2)^{\frac{3}{2}}}$

- **曲率半径**: $\rho = \frac{1}{k}$

- **曲率中心**:

  $$\alpha = x - \frac{y'(1+y'^2)}{y''}$$

  $$\beta = y + \frac{1+y'^2}{y''}$$

- **曲率圆方程**: $(x-\alpha)^2 + (y-\beta)^2 = \rho^2$

**(10) 中值定理**

- **罗尔定理 (Rolle)**: 若 $f(a) = f(b)$，则 $\exists \xi \in (a, b)$，使得 $f'(\xi) = 0$。
- **拉格朗日中值定理 (Lagrange)**: $\exists \xi \in (a, b)$，使得 $\frac{f(b)-f(a)}{b-a} = f'(\xi)$。
- **柯西中值定理 (Cauchy)**: $\exists \xi \in (a, b)$，使得 $\frac{f(b)-f(a)}{g(b)-g(a)} = \frac{f'(\xi)}{g'(\xi)}$。

**(11) 介值定理** $f(x)$ 在 $[a, b]$ 连续，且 $f(a) \neq f(b)$。 $\Rightarrow$ 对于 $C$ 满足 $f(a) < C < f(b)$ 或 $f(b) < C < f(a)$， $\exists \xi \in (a, b)$ 满足 $f(\xi) = C$。

## 7. 导数公式补充

**(12)**

$$(\tan x)' = \sec^2 x$$

$$(\cot x)' = -\csc^2 x$$

$$(\sec x)' = \sec x \tan x$$

$$(\csc x)' = -\csc x \cot x$$

$$(\arcsin x)' = \frac{1}{\sqrt{1-x^2}}$$

$$(\arccos x)' = -\frac{1}{\sqrt{1-x^2}}$$

$$(\arctan x)' = \frac{1}{1+x^2}$$

$$(\text{arccot } x)' = -\frac{1}{1+x^2}$$
