# HTML

## 1.语法糖

VS Code **内置**了对 **Emmet** 的支持，这是编写 HTML 和 CSS 最强大的“语法糖”。

- **基本用法：** 输入一个类似 CSS 选择器的缩写，然后按 `Tab` 键（或 `Enter` 键）即可快速展开成完整的 HTML 结构。
  - **快速生成 HTML5 骨架：**
    - 输入 `!` 或 `html:5`，然后按 `Tab` $\rightarrow$ 展开成一个完整的 HTML5 页面结构。
  - **生成带类的 div：**
    - 输入 `div.container`，然后按 `Tab` $\rightarrow$ 展开成 `<div class="container"></div>`。
  - **生成嵌套结构：**
    - 输入 `ul>li*5>a`，然后按 `Tab` $\rightarrow$ 展开成包含 5 个 `li` 元素的 `ul` 列表，每个 `li` 中都有一个 `a` 标签。
  - **生成带 ID 的元素：**
    - 输入 `p#intro`，然后按 `Tab` $\rightarrow$ 展开成 `<p id="intro"></p>`。
