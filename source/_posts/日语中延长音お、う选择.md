在日语中，**「お段」假名的长音**，也就是「お」音的延长，通常使用**「う」**来书写，而不是「お」。



## 📝 「お段」长音的原则





### 1. 绝大多数情况：使用「う」



当「お段」的假名（例如：お、こ、そ、と、の、ほ、も、よ、ろ、を）后面紧跟一个**「う」**时，这个「う」表示**前面的「お段」假名的长音**。

| **お段假名** | **长音写法** | **发音 (罗马字)**     | **含义**                   |
| ------------ | ------------ | --------------------- | -------------------------- |
| **こ**       | **こう**     | $k\bar{o}$ 或 $kou$   | 高、光、行、**校**（学校） |
| **と**       | **とう**     | $t\bar{o}$ 或 $tou$   | 十、遠い（远）             |
| **お**       | **おう**     | $\bar{o}$ 或 $ou$     | 多い（多）                 |
| **きょ**     | **きょう**   | $ky\bar{o}$ 或 $kyou$ | **今日**（今天）           |

> **💡 记忆要点：** 这是最常见的规则。当听到一个长 $ō$ 音时，几乎总是写成「お段假名 + **う**」。

------



### 2. 例外情况：使用「お」



只有**极少数词汇**，为了保持其原本的词源或特殊的发音习惯，会在「お段」假名后直接使用**「お」**来表示长音。

| **お段假名** | **长音写法** | **发音 (罗马字)**   | **含义**                       |
| ------------ | ------------ | ------------------- | ------------------------------ |
| **お**       | **おお**     | $\bar{o}$ 或 $oo$   | 大きい（大）、大____人（大人） |
| **と**       | **とお**     | $t\bar{o}$ 或 $too$ | **十**（十日）                 |
| **ほ**       | **ほお**     | $h\bar{o}$ 或 $hoo$ | **頬**（脸颊）                 |
| **こ**       | **こお**     | $k\bar{o}$ 或 $koo$ | 氷（冰）                       |



### 3. 外来语和拟声词



在外来语（カタカナ，片假名）中，长音通常使用**长音符号「ー」**来表示，遵循「ア段」长音的原则，不涉及「お/う」的选择问题。

- **コーヒー** ($k\bar{o}h\bar{i}$) - 咖啡
- **スポーツ** ($sup\bar{o}tsu$) - 运动

------



## 🔑 总结和选择方法



在书写「お段」长音时，您的选择标准非常明确：

1. **首选**：一律使用**「う」**（如：**とう**きょう、**こ**うえん）。
2. **特殊记忆**：只有在遇到 **「おお」**（大）和 **「とお」**（十/远）这类词源清晰的词汇时，才使用**「お」**。
