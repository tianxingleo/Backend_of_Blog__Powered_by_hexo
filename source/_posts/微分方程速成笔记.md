# 微分方程速成笔记

## PART I 概念与基础

1.  **定义**：含有自变量 $x$、未知函数 $y$ 及其各阶导数的方程。
2.  **阶**：方程中导数的**最高阶数**。
3.  **解的分类**：
    *   **通解**：含有独立任意常数的个数与方程阶数相同的解（例如一阶方程含1个 $C$，二阶含2个 $C_1, C_2$）。
    *   **特解**：不含任意常数或任意常数确定后的解。
    *   **初始条件**：用于确定通解中任意常数的条件（定解条件）。
4.  **线性微分方程**：$y$ 和它的各阶导数都是**一次**的，且系数都是 $x$ 的函数。
    *   例：$y'' - 2y = 3$ （线性）
    *   例：$y' = y^2 + 1$ （非线性，因为 $y$ 是二次）
    *   例：$y' = \sin y + 6y$ （非线性，因为 $\sin y$）
    *   例：$y' y = 1$ （非线性，相乘算二次）

## PART II 一阶微分方程

### 1. 可分离变量的微分方程

**识别**：$y' = f(x)g(y)$ 或能化为 $P(x)dx + Q(y)dy = 0$ 的形式，即 $x$ 和 $y$ 能彻底分开。

**解法**：
1.  将 $y'$ 写为 $\frac{dy}{dx}$。
2.  分离变量：$\frac{dy}{g(y)} = f(x)dx$。
3.  两边同时积分：$\int \frac{dy}{g(y)} = \int f(x)dx$。

**注意**：求解通解时，默认分母不为0，无需讨论；若求全部解，需考虑分母为0的情况（可能会丢**奇解**）。

> **【例1】** 求 $\sqrt{1-x^2}y' = \sqrt{1-y^2}$ 的通解。
>
> **解**：
> 1. 变形：$\sqrt{1-x^2}\frac{dy}{dx} = \sqrt{1-y^2}$
> 2. 分离：$\frac{dy}{\sqrt{1-y^2}} = \frac{dx}{\sqrt{1-x^2}}$
> 3. 积分：两边积分得 $\arcsin y = \arcsin x + C$
>
> *注：$y=\pm 1$ 为奇解（包络），通解中不包含，但在求通解题目中一般不写。*

### 2. 齐次方程

**识别**：$y' = g(\frac{y}{x})$，或者方程中各项关于 $x, y$ 的次数相同（例如都是三次项）。

**解法**：
1.  令 $u = \frac{y}{x}$，即 $y = ux$。
2.  求导代换：$y' = u + x\frac{du}{dx}$。
3.  代入原方程化为可分离变量方程求解，最后回代 $u = \frac{y}{x}$。

> **【例2】** 求微分方程 $\frac{dy}{dx} = \frac{y}{x} - \frac{1}{2}(\frac{y}{x})^3$，满足 $y|_{x=1}=1$ 的特解。
>
> **解**：
> 1. 令 $u = \frac{y}{x}$，则 $y = ux, \frac{dy}{dx} = u + x\frac{du}{dx}$。
> 2. 代入：$u + x\frac{du}{dx} = u - \frac{1}{2}u^3 \Rightarrow x\frac{du}{dx} = -\frac{1}{2}u^3$。
> 3. 分离：$u^{-3}du = -\frac{1}{2}x^{-1}dx$。
> 4. 积分：$-\frac{1}{2}u^{-2} = -\frac{1}{2}\ln|x| + C$。
> 5. 整理得 $\frac{1}{u^2} = \ln|x| + C'$。即 $(\frac{x}{y})^2 = \ln|x| + C'$。
> 6. 代入初值：$x=1, y=1 \Rightarrow u=1$。$1 = 0 + C' \Rightarrow C'=1$。
> 7. 特解为：$y^2 = \frac{x^2}{\ln x + 1}$ （注：因 $x=1$ 时 $y=1>0$，且定义域需满足对数要求）。

### 3. 一阶线性微分方程

**识别**：$y' + p(x)y = q(x)$。
*   $y'$ 和 $y$ 是一次，系数仅含 $x$。
*   当 $q(x) \equiv 0$ 为齐次，否则为非齐次。

**解法**：直接背公式！
$$ y = e^{-\int p(x)dx} \left[ \int q(x) e^{\int p(x)dx} dx + C \right] $$
*注：公式中 $e^{\int p(x)dx}$ 部分的不定积分不需要加绝对值和常数。*

> **【例3 几何应用】** 设某曲线 $y=f(x)$ 过点 $(1,1)$，它的切线的纵截距等于切点横坐标，求该曲线方程。
>
> **解**：
> 1. 设切点为 $(x,y)$，切线方程 $Y - y = y'(X - x)$。
> 2. 令 $X=0$，纵截距为 $y - xy'$。
> 3. 列方程：$y - xy' = x \Rightarrow y' - \frac{1}{x}y = -1$。
> 4. 识别：$p(x) = -\frac{1}{x}, q(x) = -1$。
> 5. 套公式：
>    $$ y = e^{\int \frac{1}{x}dx} \left[ \int (-1) e^{-\int \frac{1}{x}dx} dx + C \right] $$
>    $$ = e^{\ln x} \left[ -\int e^{-\ln x} dx + C \right] = x \left[ -\int \frac{1}{x} dx + C \right] $$
>    $$ = x(-\ln x + C) $$
>    *(注：因题设过(1,1)，默认 $x>0$，去绝对值)*
> 6. 代初值：$x=1, y=1 \Rightarrow 1 = 1(0+C) \Rightarrow C=1$。
> 7. 结果：$y = x(1-\ln x)$。

### 4. 积分方程

**识别**：含有变限积分 $\int_0^x f(t)dt$ 的方程。

**解法**：
1.  两边对 $x$ 求导，化为微分方程。
    *   注意：$\left( \int_0^x f(t)dt \right)' = f(x)$。
2.  **隐藏初始条件**：令积分上下限相等（如 $x=0$），使积分项为0，求出 $y(0)$ 的值。

> **【例4】** 求连续函数 $f(x)$，满足 $f(x) + 2\int_0^x f(t)dt = x^2$。
>
> **解**：
> 1. 求导：$f'(x) + 2f(x) = 2x$。这是一个一阶线性方程，$p(x)=2, q(x)=2x$。
> 2. 套公式：
>    $$ f(x) = e^{-2x} \left[ \int 2x e^{2x} dx + C \right] $$
>    $$ = e^{-2x} \left[ xe^{2x} - \frac{1}{2}e^{2x} + C \right] = x - \frac{1}{2} + Ce^{-2x} $$
> 3. 找初值：原方程中令 $x=0 \Rightarrow f(0) + 0 = 0 \Rightarrow f(0)=0$。
> 4. 定常数：$0 = - \frac{1}{2} + C \Rightarrow C = \frac{1}{2}$。
> 5. 结果：$f(x) = x - \frac{1}{2} + \frac{1}{2}e^{-2x}$。

### 5. 伯努利方程

**识别**：$y' + p(x)y = q(x)y^\alpha$ ($\alpha \neq 0, 1$)。

**解法**：
1.  **换元**：令 $z = y^{1-\alpha}$。
2.  **化简**：方程转化为关于 $z$ 的一阶线性方程：
    $$ z' + (1-\alpha)p(x)z = (1-\alpha)q(x) $$
3.  解出 $z$ 后回代。

> **【例6】** 求方程 $y' + \frac{4x}{x^2-1}y = x\sqrt{y}$ 的通解。
>
> **解**：
> 1. 识别：右边有 $\sqrt{y}=y^{1/2}$，是伯努利方程，$\alpha = 1/2$。
> 2. 换元：令 $z = y^{1-1/2} = \sqrt{y}$。
> 3. 转化：新方程为 $z' + \frac{1}{2} \cdot \frac{4x}{x^2-1}z = \frac{1}{2}x$，即 $z' + \frac{2x}{x^2-1}z = \frac{1}{2}x$。
> 4. 求解一阶线性（$z$ 关于 $x$）：
>    $$ z = e^{-\int \frac{2x}{x^2-1}dx} \left[ \int \frac{x}{2} e^{\int \frac{2x}{x^2-1}dx} dx + C \right] $$
>    $$ = e^{-\ln(x^2-1)} \left[ \int \frac{x}{2} (x^2-1) dx + C \right] $$
>    $$ = \frac{1}{x^2-1} \left[ \frac{x^4}{8} - \frac{x^2}{4} + C \right] = \frac{x^4 - 2x^2 + 8C}{8(x^2-1)} $$
> 5. 回代：$y = z^2 = \left[ \frac{x^2(x^2-2)+C'}{8(x^2-1)} \right]^2$。

## PART III 二阶可降阶微分方程

### 类型1：缺 y 型

**识别**：$y'' = f(x, y')$ （方程中不显含 $y$）。

**解法**：令 $y' = p(x)$，则 $y'' = p'$。降阶为关于 $p$ 的一阶方程。

> **【例7】** $xy'' + 3y' = 0$ 通解。
>
> **解**：
> 1. 令 $y'=p$，则 $xp' + 3p = 0$。
> 2. 分离变量：$\frac{dp}{p} = -\frac{3}{x}dx \Rightarrow \ln|p| = -3\ln|x| + C_1 \Rightarrow p = Cx^{-3}$。
> 3. 再积分：$y' = Cx^{-3} \Rightarrow y = \int Cx^{-3}dx = -\frac{C}{2}x^{-2} + C_2$。
> 4. 整理常数：$y = \frac{C_1}{x^2} + C_2$。

### 类型2：缺 x 型

**识别**：$y'' = f(y, y')$ （方程中不显含 $x$）。

**解法**：令 $y' = p(y)$，则 $y'' = p \frac{dp}{dy}$。降阶为关于 $p$ 和 $y$ 的一阶方程。

> **【例8】** $yy'' + y'^2 = 0$，满足 $y(0)=1, y'(0)=1/2$ 的特解。
>
> **解**：
> 1. 令 $y'=p$，则 $y'' = p \frac{dp}{dy}$。方程化为 $y p \frac{dp}{dy} + p^2 = 0$。
> 2. 舍去 $p=0$（不符初值），分离变量：$\frac{dp}{p} = -\frac{dy}{y} \Rightarrow p = \frac{C_1}{y}$。
> 3. 代入初值确定 $C_1$：当 $y=1$ 时 $p=1/2 \Rightarrow C_1=1/2$。故 $p = \frac{1}{2y}$。
> 4. 还原：$\frac{dy}{dx} = \frac{1}{2y} \Rightarrow 2ydy = dx$。
> 5. 积分：$y^2 = x + C_2$。
> 6. 代入初值确定 $C_2$：$x=0, y=1 \Rightarrow 1 = 0 + C_2 \Rightarrow C_2=1$。
> 7. 特解：$y = \sqrt{x+1}$ （因 $y(0)=1>0$，取正根）。

## PART IV 二阶常系数线性微分方程

形式：$y'' + py' + qy = f(x)$ （$p, q$ 为常数）。

### 1. 齐次方程 ($f(x)=0$)

**解法**：解特征方程 $\lambda^2 + p\lambda + q = 0$。
*   **两个不等实根** $\lambda_1 \neq \lambda_2$：$y = C_1 e^{\lambda_1 x} + C_2 e^{\lambda_2 x}$。
*   **两个相等实根** $\lambda_1 = \lambda_2$：$y = (C_1 + C_2 x) e^{\lambda_1 x}$。
*   **一对共轭复根** $\alpha \pm \beta i$：$y = e^{\alpha x} (C_1 \cos \beta x + C_2 \sin \beta x)$。

> **【例10】** 求 $y'' - y' + \frac{1}{4}y = 0$ 通解。
>
> **解**：特征方程 $\lambda^2 - \lambda + \frac{1}{4} = 0 \Rightarrow (\lambda - \frac{1}{2})^2 = 0 \Rightarrow \lambda_1=\lambda_2=\frac{1}{2}$。
> 通解：$y = (C_1 + C_2 x)e^{\frac{1}{2}x}$。

### 2. 非齐次方程 ($f(x) \neq 0$)

**结构**：通解 = 对应齐次通解 + 一个非齐次特解 ($y = Y + y^*$)。

**特解求法（待定系数法）**：

*   **类型一**：$f(x) = P_m(x)e^{\alpha x}$
    *   设特解 $y^* = x^k Q_m(x) e^{\alpha x}$。
    *   $k$ 的取值：看 $\alpha$ 是特征方程的几重根。
        *   不是根 $\Rightarrow k=0$
        *   单根 $\Rightarrow k=1$
        *   重根 $\Rightarrow k=2$
    *   $Q_m(x)$ 是与 $P_m(x)$ 同次的一般多项式。

*   **类型二**：$f(x) = e^{\alpha x}[P_l(x)\cos\beta x + P_n(x)\sin\beta x]$
    *   设特解 $y^* = x^k e^{\alpha x}[Q_m^{(1)}(x)\cos\beta x + Q_m^{(2)}(x)\sin\beta x]$。
    *   其中 $m = \max\{l, n\}$。
    *   $k$ 的取值：看 $\alpha \pm \beta i$ 是否为特征根。
        *   不是根 $\Rightarrow k=0$
        *   是根 $\Rightarrow k=1$

> **【例11】** 求 $y'' + 4y' + 4y = e^{-2x}$ 的通解。
>
> **解**：
> 1. **求齐次通解**：$\lambda^2 + 4\lambda + 4 = 0 \Rightarrow \lambda_{1,2} = -2$。
>    $Y = (C_1 + C_2 x)e^{-2x}$。
> 2. **求特解**：右边是 $e^{-2x}$，即 $\alpha = -2$。
>    因 $-2$ 是二重特征根，故 $k=2$。
>    设 $y^* = A x^2 e^{-2x}$。
>    求导代入原方程（计算略），解得 $A = 1/2$。
>    $y^* = \frac{1}{2}x^2 e^{-2x}$。
> 3. **总通解**：$y = (C_1 + C_2 x)e^{-2x} + \frac{1}{2}x^2 e^{-2x}$。

> **【例12】** 求 $y'' - 2y' + 5y = -\frac{1}{2}e^x \cos 2x$ 的通解。
>
> **解**：
> 1. **求齐次通解**：$\lambda^2 - 2\lambda + 5 = 0 \Rightarrow \lambda = 1 \pm 2i$。
>    $\alpha=1, \beta=2$。
>    $Y = e^x(C_1 \cos 2x + C_2 \sin 2x)$。
> 2. **求特解**：右边对应 $\alpha=1, \beta=2$，即 $\alpha \pm \beta i = 1 \pm 2i$ 正好是特征根，故 $k=1$。
>    设 $y^* = x e^x (A \cos 2x + B \sin 2x)$。
>    *(注意：虽然右边只有cos，特解里sin和cos都要设)*
>    代入原方程比较系数解得 $A=0, B=1/8$（此为示例结果，具体需计算）。
>    $y^* = \frac{1}{8} x e^x \sin 2x$。
> 3. **总通解**：$y = e^x(C_1 \cos 2x + C_2 \sin 2x) + \frac{1}{8} x e^x \sin 2x$。

### 叠加原理

若方程右边是两项之和 $f(x) = f_1(x) + f_2(x)$，则特解 $y^* = y_1^* + y_2^*$，分别针对 $f_1$ 和 $f_2$ 求特解再相加。

## PART V 高阶与欧拉方程（详细版）

这一部分主要处理两类问题：一是阶数 $n \ge 3$ 的常系数线性方程，二是虽然是变系数但有特定结构的欧拉方程。

### 一、 高阶常系数线性齐次微分方程

**1. 定义与核心思路**
*   **形式**：$y^{(n)} + a_{n-1}y^{(n-1)} + \dots + a_1 y' + a_0 y = 0$（$a_i$ 为常数）。
*   **解法**：与二阶完全一致，核心是解 **$n$ 次特征方程**：
    $$ \lambda^n + a_{n-1}\lambda^{n-1} + \dots + a_1 \lambda + a_0 = 0 $$
    虽然高次方程难解，但考试通常会给出可以因式分解的形式。

**2. 特征根与通解的对应关系表**

| 特征根的情况                        | 通解中对应的项 (线性无关解)                                  |
| :---------------------------------- | :----------------------------------------------------------- |
| **单实根** $\lambda$                | $C e^{\lambda x}$                                            |
| **$k$ 重实根** $\lambda$            | $(C_1 + C_2 x + \dots + C_k x^{k-1})e^{\lambda x}$           |
| **单复根** $\alpha \pm \beta i$     | $e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x)$          |
| **$k$ 重复根** $\alpha \pm \beta i$ | $e^{\alpha x}[(C_1 + \dots + C_k x^{k-1})\cos \beta x + (D_1 + \dots + D_k x^{k-1})\sin \beta x]$ |

---

> **【例13】** 求 $y^{(4)} - 2y''' + 5y'' = 0$ 的通解。
>
> **【解题过程】**
>
> **第一步：写出特征方程**
> 将导数阶数换成 $\lambda$ 的次数：
> $$ \lambda^4 - 2\lambda^3 + 5\lambda^2 = 0 $$
>
> **第二步：因式分解求特征根**
> 提取公因式 $\lambda^2$：
> $$ \lambda^2 (\lambda^2 - 2\lambda + 5) = 0 $$
>
> 由此得到两组根：
> 1.  $\lambda^2 = 0 \Rightarrow \lambda_1 = \lambda_2 = 0$。这是 **二重实根 0**。
> 2.  $\lambda^2 - 2\lambda + 5 = 0$。由求根公式或配方 $(\lambda-1)^2 = -4$，解得 $\lambda_{3,4} = 1 \pm 2i$。这是 **一对共轭复根**（$\alpha=1, \beta=2$）。
>
> **第三步：拼写通解**
> *   对应二重根 $0$ 的部分：$(C_1 + C_2 x)e^{0x} = C_1 + C_2 x$。
> *   对应复根 $1 \pm 2i$ 的部分：$e^{1\cdot x}(C_3 \cos 2x + C_4 \sin 2x)$。
>
> **最终结果**：
> $$ y = C_1 + C_2 x + e^x(C_3 \cos 2x + C_4 \sin 2x) $$

### 二、 欧拉方程

这是考研及高数竞赛中唯一需要掌握的一类**高阶变系数**线性微分方程。

**1. 识别题型**
*   **特征**：每一项中，**$x$ 的次数**与 **导数的阶数** 严格一致。
*   **标准形式**：
    $$ x^n y^{(n)} + \dots + a_2 x^2 y'' + a_1 x y' + a_0 y = f(x) $$

**2. 解法：算子法（Operator Method）**
相比于死记硬背代换公式，使用微分算子 $D$ 更容易推导和记忆。

*   **换元**：令 $x = e^t$ （默认 $x>0$），则 $t = \ln x$。
    我们将方程从关于 $x$ 的变系数方程，转化为关于 $t$ 的**常系数**方程。
*   **引入算子**：记 $D = \frac{d}{dt}$ （注意是对 $t$ 求导）。
*   **代换规则（口诀：阶数是多少，就乘到 $D$ 减几）**：
    *   $y$ 保持不变 $\to y$
    *   $x y'$ 换成 $\to Dy$
    *   $x^2 y''$ 换成 $\to D(D-1)y$
    *   $x^3 y'''$ 换成 $\to D(D-1)(D-2)y$
    *   以此类推...

**3. 固定步骤**
1.  令 $x = e^t$。
2.  将原方程中的 $x^k y^{(k)}$ 全部替换为 $D$ 算子形式。
3.  化简得到关于 $y(t)$ 的常系数线性微分方程。
4.  求解出 $y = f(t)$。
5.  **回代**：将 $t$ 换回 $\ln x$，将 $e^t$ 换回 $x$。

---

> **【例14】** 求欧拉方程 $x^2 \frac{d^2y}{dx^2} + 4x \frac{dy}{dx} + 2y = 0 \ (x>0)$ 的通解。
>
> **【解题过程】**
>
> **第一步：换元与代入算子**
> 令 $x = e^t$ ($t = \ln x$)，记 $D = \frac{d}{dt}$。
> 原方程逐项替换：
> *   $x^2 y'' \to D(D-1)y$
> *   $4x y' \to 4Dy$
> *   $2y \to 2y$
>
> 代入方程得：
> $$ D(D-1)y + 4Dy + 2y = 0 $$
>
> **第二步：整理为常系数方程**
> 提取 $y$，整理算子多项式：
> $$ (D^2 - D + 4D + 2)y = 0 $$
> $$ (D^2 + 3D + 2)y = 0 $$
>
> 这实际上就是关于 $t$ 的二阶常系数齐次方程：
> $$ \frac{d^2y}{dt^2} + 3\frac{dy}{dt} + 2y = 0 $$
>
> **第三步：求解常系数方程**
> 写出特征方程：
> $$ \lambda^2 + 3\lambda + 2 = 0 $$
> 因式分解：$(\lambda + 1)(\lambda + 2) = 0$
> 特征根：$\lambda_1 = -1, \lambda_2 = -2$
>
> 写出关于 $t$ 的通解：
> $$ y = C_1 e^{-t} + C_2 e^{-2t} $$
>
> **第四步：回代整理**
> 因为 $x = e^t$，所以：
> *   $e^{-t} = \frac{1}{e^t} = \frac{1}{x}$
> *   $e^{-2t} = \frac{1}{(e^t)^2} = \frac{1}{x^2}$
>
> 代入通解表达式：
> $$ y = C_1 \cdot \frac{1}{x} + C_2 \cdot \frac{1}{x^2} $$
>
> **最终结果**：
> $$ y = \frac{C_1}{x} + \frac{C_2}{x^2} $$
