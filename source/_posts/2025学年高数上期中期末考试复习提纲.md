# 导数与微分中值定理



## 第一部分：高阶导数 (High-Order Derivatives)



### 1. 常见函数的高阶导数公式



*(对应图片 115434.png)*

1. **指数函数：**
   - $(a^x)^{(n)} = a^x (\ln a)^n$
   - $(e^{ax})^{(n)} = a^n e^{ax}$
2. **分式函数：**
   - $\left(\frac{1}{ax+b}\right)^{(n)} = \frac{(-1)^n n! a^n}{(ax+b)^{n+1}}$
3. **正弦函数：**
   - $[\sin(ax+b)]^{(n)} = a^n \sin\left(ax+b + \frac{n\pi}{2}\right)$
   - $(\sin x)^{(n)} = \sin\left(x + \frac{n\pi}{2}\right)$
4. **余弦函数：**
   - $[\cos(ax+b)]^{(n)} = a^n \cos\left(ax+b + \frac{n\pi}{2}\right)$
   - $(\cos x)^{(n)} = \cos\left(x + \frac{n\pi}{2}\right)$
5. **对数函数：**
   - $[\ln(ax+b)]^{(n)} = \frac{(-1)^{n-1} a^n (n-1)!}{(ax+b)^n}$
6. **幂函数：**
   - $(x^\alpha)^{(n)} = \alpha(\alpha-1)\cdots(\alpha-n+1)x^{\alpha-n}$
   - **特例：**
     - $(x^n)^{(n)} = n!$
     - $(x^k)^{(n)} = 0 \quad (k < n, k \in \mathbb{N}^*)$



### 2. 乘积的高阶导数：莱布尼茨公式 (Leibniz Formula)



*(对应图片 115646.png)*

若 $u, v$ 均 $n$ 阶可导，则：



$$(uv)^{(n)} = \sum_{k=0}^{n} C_n^k u^{(k)} v^{(n-k)}$$

展开形式：



$$(uv)^{(n)} = u v^{(n)} + C_n^1 u' v^{(n-1)} + C_n^2 u'' v^{(n-2)} + \dots + C_n^{n-1} u^{(n-1)} v' + u^{(n)} v$$

## 第二部分：泰勒公式 (Taylor's Formula)





### 1. 带拉格朗日余项的泰勒公式



*(对应图片 121824.png)*

$$f(x) = f(x_0) + \frac{f'(x_0)}{1!}(x-x_0) + \frac{f''(x_0)}{2!}(x-x_0)^2 + \dots + \frac{f^{(n)}(x_0)}{n!}(x-x_0)^n + R_n(x)$$

拉格朗日余项：



$$R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-x_0)^{n+1} \quad (\xi \text{ 在 } x_0 \text{ 与 } x \text{ 之间})$$



### 2. 五个重要的麦克劳林展开式 ($x_0=0$)



*(对应图片 121858.png)*

通用形式：$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(0)}{n!} x^n$

1. $e^x$：

   

   $$e^x = 1 + x + \frac{x^2}{2!} + \dots + \frac{x^n}{n!} + o(x^n)$$

2. $\sin x$ (只有奇数项)：

   

   $$\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \dots + \frac{(-1)^n x^{2n+1}}{(2n+1)!} + o(x^{2n+2})$$

3. $\cos x$ (只有偶数项)：

   

   $$\cos x = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \dots + \frac{(-1)^n x^{2n}}{(2n)!} + o(x^{2n+1})$$

4. $\ln(1+x)$：

   

   $$\ln(1+x) = x - \frac{x^2}{2} + \frac{x^3}{3} - \dots + \frac{(-1)^{n-1} x^n}{n} + o(x^n)$$

5. $(1+x)^\alpha$ (二项式展开)：

   

   $$(1+x)^\alpha = 1 + \alpha x + \frac{\alpha(\alpha-1)}{2!} x^2 + \dots + \frac{\alpha(\alpha-1)\cdots(\alpha-n+1)}{n!} x^n + o(x^n)$$

   - **常见特例：**
     - $\sqrt{1+x} = 1 + \frac{1}{2}x - \frac{1}{8}x^2 + o(x^2)$
     - $\frac{1}{1+x} = 1 - x + x^2 - x^3 + \dots$
     - $\frac{1}{1-x} = 1 + x + x^2 + x^3 + \dots$



### 3. 应用：用泰勒公式求高阶导数



*(对应图片 122206.png)*

核心思想： 利用泰勒展开式的唯一性，对比系数求导数。

泰勒展开的通项系数为 $\frac{f^{(n)}(0)}{n!}$，若能写出 $x^n$ 的系数 $a_n$，则 $f^{(n)}(0) = a_n \cdot n!$。

**例题：** 求 $f(x) = x^2 \ln(1+x)$ 在 $x=0$ 处的 $n$ 阶导数 $f^{(n)}(0) \quad (n \ge 3)$。

**解题步骤：**

1. 展开 $\ln(1+x)$：

   

   $$\ln(1+x) = x - \frac{1}{2}x^2 + \dots + \frac{(-1)^{n-3}}{n-2}x^{n-2} + \frac{(-1)^{n-2}}{n-1}x^{n-1} + \dots$$

2. 代入 $f(x)$ 并乘进 $x^2$：

   

   $$f(x) = x^2 \left[ \dots + \frac{(-1)^{n-3}}{n-2}x^{n-2} + \dots \right] = \dots + \frac{(-1)^{n-3}}{n-2}x^n + \dots$$

3. 由泰勒展开式唯一性，对应 $x^n$ 的系数：

   

   $$\frac{f^{(n)}(0)}{n!} = \frac{(-1)^{n-3}}{n-2} = \frac{(-1)^{n-1}}{n-2}$$

4. 结论：

   

   $$f^{(n)}(0) = \frac{(-1)^{n-1} n!}{n-2}$$

> **注：** 泰勒公式仅能求**一点处**的导数值 $f^{(n)}(x_0)$，不可求导函数表达式 $f^{(n)}(x)$。

## 第三部分：微分中值定理 (Differential Mean Value Theorems)





### 1. 四大定理体系



*(对应图片 115902.png)*

| **定理**              | **条件**                                                | **结论**                                                     |
| --------------------- | ------------------------------------------------------- | ------------------------------------------------------------ |
| **费马引理** (Fermat) | $f(x)$ 在可导点 $x_0$ 处取极值                          | $f'(x_0) = 0$                                                |
| **罗尔定理** (Rolle)  | ① $[a,b]$ 连续 ② $(a,b)$ 可导 ③ $f(a)=f(b)$             | 存在 $\xi \in (a,b)$，使 $f'(\xi) = 0$                       |
| **拉格朗日中值定理**  | ① $[a,b]$ 连续 ② $(a,b)$ 可导                           | 存在 $\xi \in (a,b)$，使 $f(b)-f(a) = f'(\xi)(b-a)$          |
| **柯西中值定理**      | ① $f, g$ 在 $[a,b]$ 连续 ② $(a,b)$ 可导 ③ $g'(x) \ne 0$ | 存在 $\xi \in (a,b)$，使 $\frac{f(b)-f(a)}{g(b)-g(a)} = \frac{f'(\xi)}{g'(\xi)}$ |

# 三角函数

### 1. 倒数与商数关系（基础铺垫）



*这些在化简被积函数时最常用，比如把 $\tan$ 拆成 $\sin/\cos$。*

- $\tan x = \frac{\sin x}{\cos x}$
- $\cot x = \frac{\cos x}{\sin x} = \frac{1}{\tan x}$
- $\sec x = \frac{1}{\cos x}$
- $\csc x = \frac{1}{\sin x}$



### 2. 平方关系（勾股性质）



*解不定积分时，常用于凑微分或三角代换（如 $x=a\tan t$）。*

1. **$\sin^2 x + \cos^2 x = 1$** （万恶之源，最常用）
2. **$1 + \tan^2 x = \sec^2 x$** （**重点**：求 $\int \sec^2 x dx$ 或 $\int \tan^2 x dx$ 时必用）
3. $1 + \cot^2 x = \csc^2 x$

------



### 3. 倍角公式（简化计算的核心）



*在求极限和化简导数时非常有用。*

- 正弦倍角：

  

  $$\sin 2x = 2\sin x \cos x$$

- 余弦倍角（有三种形式，灵活选择）：

  

  $$\cos 2x = \cos^2 x - \sin^2 x$$

  $$\cos 2x = 2\cos^2 x - 1$$

  $$\cos 2x = 1 - 2\sin^2 x$$

- 正切倍角：

  

  $$\tan 2x = \frac{2\tan x}{1 - \tan^2 x}$$

------



### 4. 降幂公式（积分的救命稻草） ★★★



*这是高数中**最重要**的一组变形！遇到 $\sin^2 x$ 或 $\cos^2 x$ 的积分，必须降幂才能积出来。*

- 正弦降幂：

  

  $$\sin^2 x = \frac{1 - \cos 2x}{2}$$

- 余弦降幂：

  

  $$\cos^2 x = \frac{1 + \cos 2x}{2}$$

> **记忆口诀：** “正弦减，余弦加”（看分子中间的符号）。

------



### 5. 和差角公式（展开项）



*用于证明导数公式或处理复合角的极限。*

- $\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta$
- $\cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta$ （**注意符号相反**）
- $\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta}$

------



### 6. 积化和差与和差化积（积分难点）



*当你遇到 $\int \sin 3x \cos 5x \, dx$ 这种“不同频率三角函数相乘”的积分时，**必须**用积化和差把它们拆开。*



#### 积化和差（重点掌握）：



$$\sin A \cos B = \frac{1}{2} [\sin(A+B) + \sin(A-B)]$$

$$\cos A \sin B = \frac{1}{2} [\sin(A+B) - \sin(A-B)]$$

$$\cos A \cos B = \frac{1}{2} [\cos(A+B) + \cos(A-B)]$$

$$\sin A \sin B = -\frac{1}{2} [\cos(A+B) - \cos(A-B)]$$

 （**注意前面的负号**）



#### 和差化积（了解即可，高数用得少）：



$$\sin A + \sin B = 2\sin\frac{A+B}{2}\cos\frac{A-B}{2}$$

$$\cos A + \cos B = 2\cos\frac{A+B}{2}\cos\frac{A-B}{2}$$



(其余略，通常可以通过逆运算推导)

------



### 7. 辅助角公式（合并同类项）



*当遇到 $a\sin x + b\cos x$ 时，将其合并为一个正弦函数，方便求极值或周期。*

$$a\sin x + b\cos x = \sqrt{a^2+b^2} \sin(x + \phi)$$



其中辅助角 $\phi$ 满足 $\tan \phi = \frac{b}{a}$。

------



### 8. 万能公式（有理函数积分专用）



*主要用于计算“三角有理式”的积分，即万能代换 $u = \tan \frac{x}{2}$。*

设 $u = \tan \frac{x}{2}$，则：



$$\sin x = \frac{2u}{1+u^2}$$

$$\cos x = \frac{1-u^2}{1+u^2}$$

$$\tan x = \frac{2u}{1-u^2}$$

$$dx = \frac{2}{1+u^2} du$$

------



### 9. 极限中的等价无穷小（泰勒展开前几项）



*在 $x \to 0$ 时使用，秒杀极限题。*

- $\sin x \sim x$
- $\tan x \sim x$
- $\arcsin x \sim x$
- $\arctan x \sim x$
- $1 - \cos x \sim \frac{1}{2}x^2$  （**这个超级重要**）
- $\tan x - \sin x \sim \frac{1}{2}x^3$ （由泰勒公式推导，常用结论）

------



### 10. 重要的反三角函数恒等式



*在求导和积分定义域判断中会用到。*

- $\arcsin x + \arccos x = \frac{\pi}{2}$  $(-1 \le x \le 1)$
- $\arctan x + \text{arccot } x = \frac{\pi}{2}$ $(-\infty < x < +\infty)$





# 导数应用与积分基础





## 第一部分：概念辨析——什么“点”是坐标？什么“点”是数值？



*(对应图片 203101.png, 203108.png)*

这是做填空题最容易丢分的地方，务必区分清楚回答格式。



### 1. “非点”类：其实只是一个数 ($x$值)



虽然名字带“点”，但在数学定义上指的是定义域上的位置（横坐标）。**回答时只需写 $x$ 等于多少。**

- **驻点 (Stationary Point)**
  - 定义：$f'(x) = 0$ 的 $x$ 值。
  - **回答格式：** $x = x_0$
- **极值点 (Extremum Point)**
  - 定义：函数取得极大值或极小值所对应的 $x$ 值。
  - **回答格式：** $x = x_0$
  - *辨析：问“极值”答 $y$ 值；问“极值点”答 $x$ 值。*
- **间断点 (Discontinuity Point)**
  - 定义：函数不连续的地方（如分母为0的点）。
  - **回答格式：** $x = x_0$
- **不可导点**
  - **回答格式：** $x = x_0$



### 2. “为点”类：必须写成坐标形式 $(x, y)$



指的是曲线（图像）上的一个几何点，必须由横纵坐标共同确定。

- **拐点 (Inflection Point)**
  - 定义：曲线凹凸性发生改变的点。
  - **回答格式：** **$(x_0, f(x_0))$**
  - **严重警告：** 很多同学求出了 $x_0$ 直接填 $x=x_0$，这在严格评分中通常是**不得分**的。必须代回原函数求出 $y_0$，写成坐标对。

------



## 第二部分：拐点的判定与推广



*(对应图片 203154.png)*



### 1. 基础判定法



- **第一充分条件：** $f(x)$ 在 $x_0$ 处连续，且 $f''(x)$ 在 $x_0$ **两侧反号**（一侧正一侧负）。
- **第二充分条件：** $f''(x_0) = 0$ 且 $f'''(x_0) \neq 0$。



### 2. 高阶导数推广判定（奇偶法则）



设在 $x_0$ 处 $n$ 阶可导，且满足：



$$f'(x_0) = f''(x_0) = \dots = f^{(n-1)}(x_0) = 0, \quad f^{(n)}(x_0) \neq 0$$

即前 $n-1$ 阶导数全为 0，第 $n$ 阶不为 0：

- 若 $n$ 为 **奇数** (3, 5, ...) $\rightarrow$ $(x_0, f(x_0))$ 是 **拐点**。
- 若 $n$ 为 **偶数** (2, 4, ...) $\rightarrow$ $x = x_0$ 是 **极值点**。
  - ($f^{(n)} > 0$ 极小值，$f^{(n)} < 0$ 极大值)

------



## 第三部分：曲率与曲率圆



*(对应图片 203324.png)*



### 1. 曲率计算公式 ($K$)



- 直角坐标形式 ($y=f(x)$)：

  

  $$K = \frac{|y''|}{(1 + y'^2)^{3/2}}$$

- 参数方程形式 ($\begin{cases} x=x(t) \\ y=y(t) \end{cases}$):

  

  $$K = \frac{|x'(t)y''(t) - x''(t)y'(t)|}{(x'^2(t) + y'^2(t))^{3/2}}$$



### 2. 曲率半径 ($\rho$)



$$\rho = \frac{1}{K}$$



### 3. 曲率中心 $(\alpha, \beta)$ 与曲率圆



- 曲率中心坐标：

  

  $$\alpha = x - \frac{y'(1+y'^2)}{y''}, \quad \beta = y + \frac{1+y'^2}{y''}$$

- 曲率圆方程：

  

  $$(x - \alpha)^2 + (y - \beta)^2 = \rho^2$$

## 第四部分：积分“滚瓜烂熟”表



*(对应图片 204025.png, 204031.png)*

**注意：不定积分千万别忘了 $+C$ ！**



### 1. 基础函数积分



1. $\int k \, dx = kx + C$
2. $\int x^a \, dx = \frac{x^{a+1}}{a+1} + C \quad (a \neq -1)$
3. $\int \frac{1}{x} \, dx = \ln|x| + C$  **(注意绝对值)**
4. $\int \frac{1}{1+x^2} \, dx = \arctan x + C$
5. $\int \frac{1}{\sqrt{1-x^2}} \, dx = \arcsin x + C$
6. $\int a^x \, dx = \frac{a^x}{\ln a} + C$



### 2. 三角函数积分（核心组）



1. $\int \cos x \, dx = \sin x + C$
2. $\int \sin x \, dx = -\cos x + C$
3. $\int \sec^2 x \, dx = \tan x + C$  **(常用)**
4. $\int \csc^2 x \, dx = -\cot x + C$
5. $\int \sec x \tan x \, dx = \sec x + C$
6. $\int \csc x \cot x \, dx = -\csc x + C$



### 3. 容易忘的积分（对数形式）



1. $\int \sec x \, dx = \ln|\sec x + \tan x| + C$
2. $\int \csc x \, dx = \ln|\csc x - \cot x| + C$







## 第五部分：考前必背积分公式（进阶组）



*(对应图片 204305.png)*



### 1. 三角函数补充（对数型）



这两个公式通过换元法（凑微分）推导而来，建议直接背诵结论。

1. 正切函数：

   

   $$\int \tan x \, dx = -\ln|\cos x| + C$$

   

   (记忆：$\tan = \sin/\cos$，$\cos$ 在分母求导是 $-\sin$，所以有负号)

2. 余切函数：

   

   $$\int \cot x \, dx = \ln|\sin x| + C$$

   

   (记忆：$\cot = \cos/\sin$，$\sin$ 在分母求导是 $\cos$，所以是正号)

------



### 2. 含参数 $a$ 的重要积分（易混淆点）



这组公式是基础反三角函数积分的推广形式 $(a>0)$。

⚠️ 重点注意： 哪些公式前面有系数 $\frac{1}{a}$，哪些没有！

1. 反正切推广（有系数）：

   

   $$\int \frac{dx}{x^2 + a^2} = \frac{1}{a} \arctan \frac{x}{a} + C$$

   

   (易错点：千万别忘了前面的 $\frac{1}{a}$)

2. 反正弦推广（无系数）：

   

   $$\int \frac{dx}{\sqrt{a^2 - x^2}} = \arcsin \frac{x}{a} + C$$

   

   (记忆：$\arcsin$ 也就是“罪”，罪人是不需要穿这一件 $\frac{1}{a}$ 的衣服的)

------



### 3. “长对数”与有理函数积分



这两类积分形式复杂，但经常出现在分母根式的积分题中。

1. “长对数”公式：

   

   $$\int \frac{dx}{\sqrt{x^2 \pm a^2}} = \ln|x + \sqrt{x^2 \pm a^2}| + C$$

   

   (无论根号里是加还是减，对数里面都是 $x$ 加上那个根式)

2. 平方差分式（有理函数）：

   

   $$\int \frac{1}{a^2 - x^2} \, dx = \frac{1}{2a} \ln \left| \frac{a+x}{a-x} \right| + C$$

   

   (记忆：这是部分分式分解的结果 $\frac{1}{a^2-x^2} = \frac{1}{2a}(\frac{1}{a-x} + \frac{1}{a+x})$)

------



### 💡 避坑指南：系数 $\frac{1}{a}$ 的规律



在上述带 $a$ 的公式中，为了防止记混，可以遵循以下规律：

- **生成 $\arctan$ 和 $\ln$ (有理分式) 的：** 前面**有** $\frac{1}{a}$ 或 $\frac{1}{2a}$。
  - (公式 17, 20)
- **生成 $\arcsin$ 和 $\ln$ (长对数) 的：** 前面**没有** $\frac{1}{a}$。
  - (公式 18, 19)
